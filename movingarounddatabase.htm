Moving around the database is as simple as pushing a button.  Basically, there are four buttons you can use to move around the database.

    Move First: Pushing this button will take you to the very first town listed in the database.  This first town name will be listed in the Town Name textbox.  By default, when you first open the Edit Town Names Window the database starts at this position.

    Move Prev: Pushing this button will take you to the previous town in the database.  In other words, if you're at the fifth town name in the database, pushing this button will move you to the fourth town in the database. The new town name will be presented in the Town Name textbox.  If you are at the very first town name in the database you'll receive an error message telling you that you are at the first town in the database.

    Move Next: Pushing this button will take you to the next town name in the database. If you are at the first town name it will move you to the second.  The new town name will be presented in the Town Name textbox.  If you are at the very last town name in the database you'll receive an error message telling you that you are at the last town in the database.

    Move Last: Pushing this button will take you to the very last town listed in the database.  This last town name will be listed in the Town Name textbox.

You can also move around in the database by using the Find button.
