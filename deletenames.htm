Occasionally there comes a time when you might need to remove a town name from the database. This might happen when, heaven forbid, a county is removed from your ADI.  Town Chooser Version 2.0 now allows you to delete those community names you no longer wish to use.

The first thing you'll need to do is find the town name you want to delete.  Once you've found it and it's displayed in the Town Name textbox click the Delete button.

Now you should see the Delete Record dialog box.

Make sure you are absolutely sure that you want to delete this town name before you click the OK button. If you're not sure, choose the Cancel button and the town name will not be deleted. You can now press the Restore button and the database will be returned to its original condition.

If you click the OK button the town name in the Town Name textbox will be deleted and can not be recovered. If you want that town back in the database you'll have to Add it back.
